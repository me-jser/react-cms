基于React + React-router + webpack 的cms dashboard.
___

目前计划先实现后台管理部分,后期加入NodeJs和mongoDB

HOWTO
___
- 执行 `npm install` (建议使用 [cnpm](https://npm.taobao.org/))
- 执行 `npm run dev` 进行编译和watch
- 在浏览器中访问即可
